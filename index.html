// how it works

Three steps,
zero effort

01

Item Added

Jellyfin fires an event the moment a new media file lands in your library.

02

Title Unmatched

If no scraper has set a proper title, the item still has the raw filename. The plugin detects this automatically.

03

Clean & Apply

Quality tags, codecs, site watermarks, and release flags are stripped. The readable title is saved back to the item.

// installation

Up in two steps

1

Add the repository

In Jellyfin go to Dashboard → Plugins → Repositories and add:

https://adielsa.github.io/jellyfin-filename-title/manifest.json

Save and restart Jellyfin.

2

Install from Catalog

Go to Dashboard → Plugins → Catalog, find Filename Title, click Install, and restart when prompted.
